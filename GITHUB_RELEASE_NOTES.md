# MonsTranslate v1.0.0

MonsTranslate, Windows icin canli ceviri ve OCR overlay aracidir. Bu ilk paket surumu, tek dosya exe ve installer olarak dagitima hazirdir.

## Indirme

Normal kullanici icin onerilen dosya:

- `MonsTranslate_Setup.exe`

Portable kullanim icin:

- `MonsTranslate.exe`

## Ozellikler

- Canli yazma cevirisi
- Secili metin cevirisi
- EasyOCR destekli ekran metni okuma
- Sabit OCR alani secme
- Google Ucretsiz, MyMemory Ucretsiz ve Google AI Studio ceviri motorlari
- API key ekleme ve test paneli
- Kisayol duzenleme
- Sistem tepsisi destegi
- Kaydirilabilir kontrol paneli

## Varsayilan Kisayollar

- `Ctrl + Space`: Yazilan metni ceviriyle degistirir
- `Ctrl + Q`: OCR ile metni okuyup cevirir
- `Ctrl + Shift + Q`: OCR alani secimi
- `Ctrl + Shift + E`: Uygulamadan cikis

## Notlar

- Ilk acilis EasyOCR ve Torch paketleri nedeniyle biraz yavas olabilir.
- Program imzali olmadigi icin Windows Defender ilk calistirmada uyari gosterebilir.
- OCR kullanimi ekran goruntusu aldigi icin anti-cheat kullanan oyunlarda OCR kapali tutulmalidir.
