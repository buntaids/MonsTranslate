## Ilk Git Komutlari

Repo GitHub'da olusturulduktan sonra:

```powershell
git init
git add .
git commit -m "Initial MonsTranslate release"
git branch -M main
git remote add origin https://github.com/KULLANICI_ADI/MonsTranslate.git
git push -u origin main
```

Release yuklemek icin GitHub web arayuzu daha kolaydir:

1. GitHub reposuna girin.
2. `Releases` bolumunu acin.
3. `Create a new release` secin.
4. Tag: `v1.0.0`
5. Title: `MonsTranslate v1.0.0`
6. `GITHUB_RELEASE_NOTES.md` icerigini aciklama olarak ekleyin.
7. `MonsTranslate_Setup.exe` ve isterseniz `MonsTranslate.exe` dosyasini yukleyin.
